This app allows operators to easily plan and visualize inspection missions for four-legged utility robots. The purpose is to empower operators to designate waypoints and routes for the robots to autonomously navigate and perform inspection tasks.

The app uses Angular and Angular Material for the frontend and has a three page structure:

  • Home Page - Provides an overview and build/run instructions.
  • Planner Page - Allows creating/editing robot waypoints.
  • Player Page - Visualizes and plays back the route.

The app follows consistent coding standards including:

  • 168 character line length limit
  • CSS BEM methodology
  • Route/feature based hierarchical folder structure
